Q11.1-1
- 크기 m을 가지는 직접 주소 테이블 T를 순회해야 한다
  최악의 경우 성능은 O(m)이 된다.

Q11.1-2
- m개의 값을 비트로 변환해서, 비트 벡터에 저장하면 된다. 


Q11.1-3
- INSERT - 입력 되는 값을 직접 주소 테이블의 크기(ex) M)로 모듈로 연산을 해서, 위치를 구한 후,
           해당 위치에 연결 리스트로 값을 더한다
- DELETE - 입력 되는 값을 직접 주소 테이블의 크기(ex) M)로 모듈로 연산을 해서, 위치를 구한 후,
           해당 위치의 연결 리스트에서 값을 제거한다.
- SEARCH - 입력 되는 값을 직접 주소 테이블의 크기(ex) M)로 모듈로 연산을 해서, 위치를 구한 후,
           해당 위치의 연결 리스트에서 값을 찾는다.
